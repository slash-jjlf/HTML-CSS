<h1> ~ <h6> Heading
- 웹 페이지의 제목 또는 부제목을 표현할 때 사용. 숫자가 작을 수록 큰 제목을 표시하는데 사용

<P> Paragraph
- 하나의 문단을 표시할 때 사용

<hr> Horizontal Rule
- 가로로 선을 긋는 태그
- 종료 태그 없음

<br> Break
- 줄바꿈 태크로 HTML에서 개행 역할
- 종료 태크 없음

<i>
- 텍스트를 이텔릭체로 표시할 때 사용

<em>
- 텍스트를 이텔릭체로 강조할 때 사용

<b>
- 텍스트를 진하게 표시할 때 사용

<strong>
- 텍스트를 진하게 강조할 때 사용
