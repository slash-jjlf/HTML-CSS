<header>
- 문서의 머리말(소개 내용)을 정의. 주로 페이지의 제목, 로고, 내비게이션 링크, 검색 폼 등 포함
- 웹사이트 상단 베너, 섹션의 제목 및 부제목

<nav>
- 다른 페이지나 현재 페이지의 다른 섹션으로 연결되는 내비게이션 링크(탐색 링크)들을 정의
- 주로 내비게이션 메뉴에 사용
- 상단 메뉴바, 사이드바의 링크 목록

<main>
- 문서의 <body>내에서 메인 콘텐츠를 정의
- 각 문서에는 하나의 <main>요소만 있어야 하며, 이는 문서의 핵심적인 내용을 담음
- 블로그 게시물의 본문 내용, 제품 상세 페이지의 주요 정보

<article>
- 독립적으로 배포되거나 재사용될 수 있는, 자립적인 콘텐츠를 정의
- 블로그 게시물, 사용자 댓글 등

<section>
- 문서의 독립적인 섹션(구획)을 정의
- 일반적으로 제목(<h1> ~ <h6>)을 포함하며, 주제별로 그룹화된 콘텐츠에 사용
- <article>내에서도 여러 <section>이 있을 수 있음
- 책의 장(chapter), 웹 페이지의 소개, 문의하기 섹션

<aside>
문서의 주요 콘텐츠와 간접적으로 관련된 콘텐츠를 정의
- 사이드바, 광고 블록, 용어 설명, 관련 링크
- 블로그의 관련 글 목록, 사전 정의, 광고 배너

<footer>
- 문서 또는 섹션의 바닥글을 정의
- 주로 저작권 정보, 연락처 정보, 사이트맵, 관련 문서 링크 등이 포함
- 웹사이트 하단의 저작권 정보, 페이지 내 특정 섹션의 끝 부분
